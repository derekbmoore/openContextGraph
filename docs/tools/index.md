This section documents the **Tools** surface area used by Foundry agents and ctxEco workflows:

- **MCP (Model Context Protocol)**: our primary GTM integration surface (bring-your-own MCP server URL).
- **HTTP tool endpoints**: lightweight action-group-compatible endpoints for demos and enterprise hardening.
- **Tool governance**: allowlists, policy checks, auditing, and evidence bundles.
